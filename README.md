# vim-trex

Running T-Rex on Vim

![TREx](https://raw.githubusercontent.com/mattn/vim-trex/master/screenshot.gif)

## Usage

```
:TRex
```

## Requirements

* Electron

## Installation

```
$ npm install
```

And download T-Rex glTF model.

For Windows
```
download.cmd
```

For UNIX:
```
source download.cmd
```
